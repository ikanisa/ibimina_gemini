# Ibimina Gemini User Guide

## Overview

Ibimina Gemini is a multi-tenant group savings and loans management platform designed for SACCOs, savings groups, and microfinance institutions.

---

## Getting Started

### Login

1. Navigate to the application URL
2. Enter your email and password
3. Click "Sign In"

If you've forgotten your password:
1. Click "Forgot password?"
2. Enter your email address
3. Check your email for a reset link
4. Create a new password (must be 12+ characters with uppercase, lowercase, number, and special character)

---

## Dashboard

The dashboard provides an overview of your institution's financial activity:

- **Total Balance** - Sum of all member savings
- **Today's Deposits** - Deposits received today
- **Today's Withdrawals** - Withdrawals processed today
- **Active Members** - Number of active members

### Quick Actions
- Record a new transaction
- Add a new member
- View recent activity

---

## Members

### Viewing Members

Navigate to **Members** in the sidebar to see all registered members.

Use the search bar to find members by:
- Name
- Phone number
- Branch

### Adding a Member

1. Click **+ Add Member**
2. Fill in required fields:
   - Full Name
   - Phone Number (with country code)
   - Branch (optional)
3. Click **Save**

### Member Details

Click on any member to view:
- Personal information
- Transaction history
- Group memberships
- Savings balance
- Loan balance (if any)

---

## Transactions

### Recording Transactions

1. Go to **Transactions**
2. Click **+ New Transaction**
3. Select transaction type:
   - **Deposit** - Member adding money
   - **Withdrawal** - Member withdrawing money
   - **Transfer** - Moving funds between accounts
4. Enter amount and select payment channel
5. Click **Submit**

### Transaction Types

| Type | Description |
|------|-------------|
| DEPOSIT | Cash or mobile money deposit |
| WITHDRAWAL | Cash withdrawal |
| TRANSFER | Internal fund transfer |
| GROUP_CONTRIBUTION | Payment to a savings group |
| LOAN_REPAYMENT | Loan installment payment |

### Filtering Transactions

Use filters to narrow down transaction history:
- Date range
- Transaction type
- Status
- Member

### Exporting Data

1. Apply desired filters
2. Click **Export CSV**
3. A CSV file will download automatically

### Printing Receipts

1. Find the transaction in the list
2. Click the printer icon or **Print Receipt**
3. A printable receipt will open in a new window

---

## Groups

### Creating a Group

1. Navigate to **Groups**
2. Click **+ New Group**
3. Enter group details:
   - Group Name
   - Expected Contribution Amount
   - Frequency (Weekly/Biweekly/Monthly)
   - Grace Period (days)
4. Click **Create Group**

### Managing Group Members

1. Open the group details
2. Click **Add Member**
3. Search and select members to add

### Recording Contributions

1. Open the group
2. Click **Record Contribution**
3. Select member and enter amount
4. Click **Submit**

## Troubleshooting

### Session Expired

If you see a session expired message:
1. Your session timed out after 30 minutes of inactivity
2. Click "Log in again"
3. Re-enter your credentials

### Data Not Loading

1. Check your internet connection
2. Try the **Retry** button
3. Refresh the page
4. If issue persists, contact support

### Cannot Print Receipt

1. Ensure pop-ups are allowed for this site
2. Try a different browser
3. Use the download option instead

## Security Best Practices

1. **Use a strong password** - 12+ characters with mixed case, numbers, and symbols
2. **Don't share credentials** - Each user should have their own account
3. **Log out when done** - Especially on shared devices
4. **Report suspicious activity** - Contact your administrator immediately
