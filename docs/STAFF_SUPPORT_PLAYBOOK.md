# Staff & Support Playbook

## 1. Public Group Approvals
- **Process**:
  1. Check Admin Portal -> "Pending Groups".
  2. Call the provided leader number.
  3. Verify intent: "Are you a real savings group?"
  4. If real -> Approve.
  5. If spam/test -> Reject.

## 2. Handling Disputes
- **Scenario**: User claims they paid, Treasurer rejected it.
- **Action**:
  1. Ask User for the MTN SMS screenshot.
  2. Ask Treasurer to check their MoMo statement.
  3. If money arrived: Instruct Treasurer to "Edit Ledger" (if feature avail) or have User "Fix Submission".

## 3. Privacy Do's and Don'ts
- **DO**: Ask for "Member ID" or "Submission Ref" from the app.
- **DON'T**: Ask for full PINs.
- **DON'T**: Share screenshots of one group's ledger with another group.

## 4. WhatsApp Support Handoff
- Users will come from the "Help Center" link.
- Messages will look like: `Hi Ibimina Support... Context: help_center_home`
- If `TxID` is included, use it to search the backend logs.
