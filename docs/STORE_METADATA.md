# Store Listing Metadata

## App Details
- **App Name:** Ibimina (Gemini Edition)
- **Short Description:** Save together, grow together. Secure micro-savings for groups.
- **Full Description:**
  Ibimina is the easiest way to manage your savings group (Ikimina) digitally.
  
  **Key Features:**
  - Securely track group contributions.
  - View real-time transparency of group funds.
  - Invite members via QR code.
  - Offline-first support.
  
  **Note:** This app records transactions made via MoMo. It does not process payments directly. All money handling happens outside the app via standard USSD channels.
